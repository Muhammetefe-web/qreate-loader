## 📥 Nasıl İndirilir?
Eğer dosya direkt inmezde sizi buraya atarsa aşağıdaki talimatlara göre indirebilirsiniz:


1. Bu sayfanın **sağ tarafında** bulunan **"Releases" (Sürümler)** bölümüne bakın.
2. Oradaki en güncel sürüme (genellikle en üstte "Latest" etiketiyle bulunur) tıklayın.
3. Açılan sayfada alt kısımdaki **"Assets"** bölümünden dosyayı (örn: `qreateV2_Loader.exe` veya `.zip`) bilgisayarınıza indirin.
